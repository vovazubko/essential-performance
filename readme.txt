Essential Performance plugin aims to improve website loading performance.

== Description ==

This plugin was born with ideas to combine all the achievements that for many years he created for many projects.

This plan combines best practices to speed up the loading of both individual pages and the entire system.

With this plugin you will receive more loyalty from visitors to your site, a higher rating from search
engines and as a result even more visitors.

<h4>Features</h4>

*   Lazy Load - is a smart technique which reduces the page load time.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/essential-performance` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
